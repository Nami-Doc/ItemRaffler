ItemRaffler
===========


# Etapes d'installations
Vous avez besoin de Ruby 1.9.2 ou supérieur, ainsi que rubygems (fourni avec)
Tapez les commandes suivantes dans la console (shell, cmd.exe, ...)

 * bundle install
 * rake db:migrate
 * rails s
